2. เพศสภาพมีผลต่อแนวโน้มในการเกิดโรคหัวใจหรือไม่
3. ประเภทของโรคหัวใจที่พบมากที่สุดในผู้ป่วยชาย
4. ประเภทของโรคหัวใจที่พบมากที่สุดในผู้ป่วยหญิง
5. คอเลสเตอรอลเฉลี่ยของผู้ที่ไม่มีอาการโรคหัวใจ
